## To install, run...
> Check installation from the next section before installing

``` bash
sudo apt-get update
sudo apt-get install ros-melodic-opencv
```

## To check installation, run...

```bash
python
>>> import cv2
# if there was no error, then the installation works.
>>> exit()
```
